Máquina de Estados do Jogo:

Start State: O labirinto e o seu conteúdo são apresentados no ecrã, mas sem qualquer ação ou movimento.
Moving State: Inicia-se quando o jogador pressiona uma tecla de direção, e o Pac-Man começa a movimentar-se na direção escolhida pelo user.
//Estado dos Fantasmas: Inicia-se após 5 segundos (ou outro valor definido pelo aluno) a partir do Estado de Movimento. Os fantasmas começam a movimentar-se e o jogo entra no estado .
Vulnerable State: O jogo entra neste estado quando quando ele come uma bola de poder. O jogo entra então em um novo estado de tempo limitado, onde os fantasmas ficam vulneráveis.
Level Complete State: O nivel é completado e passa para o estado inicial com um novo nivel.
Dead State: Este estado é iniciado quando o pacman é morto, perde uma vida e se as vidas forem maiores que 0 recomeça o jogo, se forem == 0 vai para o estado de finish State
Pause State: Inicia-se quando o utilizador clicar no botao de pausa, onde poderá retomar o jogo, sair sem salvar ou sair e guardar o jogo.
Finish State: O jogo entra neste estado quando o jogador perdeu todas as suas vidas ou terminou todos os níveis.


Máquina de Estados dos Fantasmas:
Chasing State: Os fantasmas perseguem o Pac-Man.
Vulnerable State: Os fantasmas ficam vulneráveis e podem ser comidos.
Eaten State: Os fantasmas vao para as celas.
Waiting State: Os fantasmas ficam à espera para sairem da cela, ficam a agurdar até começarem a perseguir o PacMan .

NOVA MAQUINA DE ESTADOS
-Initial State   - O jogo é carregado se existir, senao é colocado um jogo novo no primeiro nivel e passa para o Start State
-Start State     - Este estado aguarda até o utilizador carregar em um botão para iniciar o jogo e passa para o freeGhost State
-lockGhost State - Este estado permite ao pacman movimentar-se mas mantem os fantasmas presos durante um determinado tempo, quando terminar esse tempo passa para o movingState
-Moving State    - Este estado permite ao pacman e aos fantasmas movimentarem-se no mapa.
	Se o pacman comer todos os alimentos vai para o "LevelCompleteState"
	Se os fantasmas comerem o Pacman, o jogador perde uma vida, se vida > 0 reinicia o jogo e volta para o "StartState", se vida == 0 o jogo termina e passa para o "FinishState"
	Se o pacman come um poder ativará um boolean e poderá comer os fantasmas fazendo com que os fantasmas fujam do pacman, quando o pacman come um fanstasma eles mudam para o "lockGhost State", após um determinado tempo 		esse boolean é desativado e os fantasmas poderão voltar a perseguir e a comer o Pacman.
	Se o jogador quiser pausar o jogo vai para o "Pause State"
-Level Complete State - Começa um novo nivel e passa para o StartState
-Paused State 		  - O jogador poderá escolher se pretende salvar o jogo e sair, ou sair do jogo sem salvar onde passa para o "Finish State"
-Finish State		  - Este estado acontece quando um jogo acaba, terminando de seguida com a maquina de estados.
